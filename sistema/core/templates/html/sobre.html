Nesta pagina estão algumas informações sobre o projeto

Sequência de eventos

1° Solicita um arquivo Excel como base

Selecione o arquivo da disciplina/turma para iniciar.

2° Escaneia o arquivo

O sistema identifica as colunas onde as notas devem ser inseridas.

3° Seleção da nota

Escolha qual nota deseja inserir (I1, I2, Prova, Recuperação).

4° Inserção dos dados

  • Nome do aluno (auto completar)
  • Instrumento avaliativo (I1, I2...)
  • Nota
  • Bimestre ou etapa

5° Recuperação automática

O sistema calcula automaticamente os alunos que estão de recuperação.

6° Análise da turma

Calcula média, aprovados e reprovados.

7° Gráfico geral

Gera gráficos com informações das matérias, recuperações e médias.

Ferramentas Utilizadas

  • HTML
  • CSS
  • JavaScript
  • Python (Django, Pandas, Pygraph)

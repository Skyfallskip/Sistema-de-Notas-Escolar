Bem-Vindo ao Sistema de Notas

Neste sistema é possivel inserir notas de alunos em um arquivo excel de forma rapida e facil!

Como funciona?

Este sistema permite que você insira notas de alunos em um arquivo Excel de maneira rápida e fácil. Basta fornecer um arquivo base e o sistema irá o escanear para saber onde inserir as notas. Ou caso prefira, é possivel gerar um diario de notas do zero.
